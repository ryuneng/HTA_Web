박스모델
    - 모든 HTML 요소는 Box 형태의 영역을 가지고 있다.
      이 Box는 컨텐츠(Content), 안쪽여백(Padding), 테두리(Border), 바깥쪽여백(Margin)으로 구성된다.

      Content    요소의 텍스트나 이미지 등의 실제 내용이 위치하는 영역
                 width, height 프로퍼티를 갖는다.

      Padding    테두리 안쪽에 위치하는 내부 여백 영역
                 Padding 프로퍼티는 Padding 영역의 두께를 의미한다.

      Border     요소의 테두리 영역
                 Border 프로퍼티는 테두리의 두께를 의미한다.

      Margin     요소의 외부 여백 영역
                 Margin 프로퍼티는 Margin 영역의 두께를 의미한다.



각 영역의 두께 지정하기
    - 4개 방향에 대해 각각 지정하기
        ex)
          div {
            margin-top: 40px;
            margin-right: 30px;
            margin-bottom: 20px;
            margin-left: 10px;
          }

    - 방향을 지정하지 않고 4방향의 프로퍼티를 한 번에 지정하기
        ex) 4개의 값을 지정할 때
          margin: 10px 20px 30px 40px;
              => margin-top: 10px;
              => margin-right: 20px;
              => margin-bottom: 30px;
              => margin-left: 40px;
              
        ex) 3개의 값을 지정할 때
          margin: 10px 20px 30px;
              => margin-top: 10px;
              => margin-right: 20px;
              => margin-left: 20px;
              => margin-bottom: 30px;

        ex) 2개의 값을 지정할 때
          margin: 10px 20px;
              => margin-top: 10px;
              => margin-bottom: 10px;
              => margin-right: 20px;
              => margin-left: 20px;

        ex) 1개의 값을 지정할 때
          margin: 10px;
              => margin-top: 10px;
              => margin-right: 10px;
              => margin-bottom: 10px;
              => margin-left: 10px;



box-sizing 프로퍼티
    - box-sizing 프로퍼티는 width, height 프로퍼티의 대상 영역을 지정한다.
    - box-sizing 프로퍼티의 기본값은 context-box다.
        context-box : width, height 프로퍼티값은 content 영역을 의미한다.
        border-box  : width, height 프로퍼티값은 content+padding+border 영역을 의미한다.
                      마진을 제외한 박스모델전체를 width, height 대상영역으로 지정할 수 있어서
                      CSS Layout을 직관적으로 사용할 수 있다.
