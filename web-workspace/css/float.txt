float 프로퍼티
    - 해당 요소를 다음 요소 위에 떠 있게(부유하게) 하는 프로퍼티
    - 본래 이미지와 텍스트가 함께 있을 때, 이미지 주위를 텍스트가 자연스럽게 감싸기 위해 만들어짐
    - float가 적용된 요소는 기본 레이아웃 흐름에서 벗어나 요소의 모서리가 페이지의 왼쪽이나 오른쪽으로 이동됨
    - 레이아웃을 구성할 때 블록 요소를 정렬하기 위해 사용되는 중요한 기법
    - float 프로퍼티의 값
        none   요소를 떠있지 않게 하는 값
        right  요소를 부유시켜서 오른쪽으로 이동시키는 값
        left   요소를 부유시켜서 왼쪽으로 이동시키는 값
